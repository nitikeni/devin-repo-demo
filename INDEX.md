## 📞 Support

### Documentation
- Check the relevant guide above
- Review the troubleshooting section
- Check browser console for errors

### Common Issues
1. **Port already in use** → Use different port: `PORT=3001 npm start`
2. **Sample data not loading** → Clear cache and try again
3. **Styles not showing** → Hard refresh (Ctrl+F5)
4. **Server won't start** → Check Node.js installation

### Getting Help
1. Read the relevant documentation
2. Check the TEST_GUIDE.md
3. Review DEPLOYMENT_GUIDE.md troubleshooting
